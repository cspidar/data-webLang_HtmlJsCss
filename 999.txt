변환 기능
이 함수는 <transform-function>CSS 데이터 유형 이 값으로 사용될 때 사용됩니다 transform.

matrix()
동종 2D 변환 행렬에 대해 설명합니다.

matrix3d()
3D 변환을 4x4 균일 행렬로 설명합니다.

perspective()
사용자와 z=0 평면 사이의 거리를 설정합니다.

rotate()
2D 평면의 고정점을 중심으로 요소를 회전합니다.

rotate3d()
3D 공간에서 고정 축을 중심으로 요소를 회전합니다.

rotatex()
가로 축을 중심으로 요소를 회전합니다.

rotatey()
수직 축을 중심으로 요소를 회전합니다.

rotatez()
Z축을 중심으로 요소를 회전합니다.

scale()
2D 평면에서 요소를 확대 또는 축소합니다.

scale3d()
3D 공간에서 요소를 확대 또는 축소합니다.

scalex()
요소의 크기를 수평으로 위 또는 아래로 조정합니다.

scaley()
요소의 크기를 수직으로 위 또는 아래로 조정합니다.

scalez()
Z축을 따라 요소의 크기를 조정합니다.

skew()
2D 평면에서 요소를 기울입니다.

skewx()
요소를 수평 방향으로 기울입니다.

skewy()
수직 방향으로 요소를 기울입니다.

translate()
2D 평면에서 요소를 변환합니다.

translate3d()
3D 공간에서 요소를 변환합니다.

translatex()
요소를 수평으로 변환합니다.

translatey()
요소를 수직으로 변환합니다.

translatez()
z축을 따라 요소를 변환합니다.

수학 함수
수학 함수를 사용하면 CSS 숫자 값을 수학 표현식으로 작성할 수 있습니다.

calc()
숫자 CSS 값에 대해 기본 산술을 수행할 수 있는 수학 함수입니다.

clamp()
최소값, 중앙값 및 최대값을 취하고 중앙 계산을 나타내는 비교 함수입니다.

max()
값 목록 중 가장 큰 값을 나타내는 비교 함수입니다.

min()
값 목록 중 가장 작은 값을 나타내는 비교 함수입니다.

abs() 
계산을 수행하고 절대값을 반환합니다.

acos() 
반환된 각도는 역삼각 함수로 [0deg, 180deg]; 범위로 정규화되어야 합니다.

asin() 
반환된 각도는 역삼각 함수로 [-90deg, 90deg] 범위로 정규화되어야 합니다.

atan() 
반환된 각도는 역삼각 함수로 [-90deg, 90deg] 범위로 정규화되어야 합니다.

atan2() 
두 개의 쉼표로 구분된 계산 A와 B를 포함합니다. A와 B는 <number>, <dimension>, 또는 로 해석될 수 <percentage>있지만 유형이 동일해야 합니다. 그렇지 않으면 함수가 유효하지 않습니다.

cos() 
<number>또는 로 해석되어야 하는 단일 계산을 포함합니다 <angle>.

exp() 
로 해석되어야 하는 하나의 계산을 포함 <number>하고 pow(e, A) 와 동일한 값을 a 로 반환합니다 <number>.

hypot() 
하나 이상의 쉼표로 구분된 계산을 포함하고 각 계산과 동일한 구성 요소가 있는 N차원 벡터의 길이를 반환합니다.

log() 
둘 다 a 로 해석되어야 <number>하고 값 A의 대수 밑 B를 a 로 반환 해야 하는 하나 또는 두 개의 계산(로그할 값과 로그의 밑을 나타냄, 기본값은 e)을 포함합니다 <number>.

mod() 
두 개의 계산 A와 B를 포함하고 A와 A 위 또는 아래에 있는 B의 가장 가까운 정수 배수 간의 차이를 반환하는 모듈러스 함수입니다.

pow() 
두 개의 쉼표로 구분된 계산 A와 B가 포함되어 있으며 둘 다 a 로 확인되어야 하며 <number>A를 B의 거듭제곱으로 제곱한 결과를 반환하고 값을 a로 반환합니다 <number>.

rem() 
두 개의 계산 A와 B를 포함하고 A와 A 위 또는 아래에 있는 B의 가장 가까운 정수 배수 간의 차이를 반환하는 모듈러스 함수입니다.

round() 
선택적 반올림 전략과 두 개의 계산 A 및 B를 포함하고 반올림 전략에 따라 반올림된 A 값을 A 위 또는 아래에 있는 B의 가장 가까운 정수 배수로 반환합니다.

sign() 
계산을 수행하고 숫자 값이 음수이면 -1, 숫자 값이 양수이면 +1, 숫자 값이 0⁺이면 0⁺, 숫자 값이 0⁻이면 0⁻를 반환합니다.

sin() 
<number>또는 로 해석되어야 하는 단일 계산을 포함합니다 <angle>.

sqrt() 
로 해석되어야 하는 단일 계산을 포함 <number>하고 값의 제곱근을 로 반환합니다 <number>.

tan() 
<number>또는 로 해석되어야 하는 단일 계산을 포함합니다 <angle>.

색상 기능
다른 색상 표현을 지정하는 함수. <color>유효한 모든 곳에서 사용할 수 있습니다 .

color() 
(대부분의 다른 색상 기능이 작동하는 암시적 sRGB 색상 공간이 아닌) 특정 지정된 색상 공간에서 색상을 지정할 수 있습니다.

color-mix() 
두 color값을 취해 주어진 색상 공간에서 주어진 양만큼 혼합한 결과를 반환합니다.

color-contrast() 
color값을 가져와 다른 color값 목록과 비교하여 목록에서 대비가 가장 높은 값을 선택합니다.

device-cmyk() 
장치 독립적인 방식으로 CMYK 색상을 표현하는 데 사용됩니다.

hsl()
HSL 색상 모델은 색상, 채도 및 밝기 구성 요소에 따라 주어진 색상을 정의합니다. 선택적 알파 구성 요소는 색상의 투명도를 나타냅니다.

hsla()
HSL 색상 모델은 색상, 채도 및 밝기 구성 요소에 따라 주어진 색상을 정의합니다. 알파 구성 요소는 색상의 투명도를 나타냅니다.

hwb() 
HWB(Hue-Whiteness-Blackness의 줄임말)는 HSL과 유사한 색상을 지정하는 또 다른 방법입니다.

lab() 
실험실 색상은 장치 독립적이며 색상의 물리적 측정값을 지정합니다.

lch() 
LCH 색상은 장치 독립적이며 채도 및 색조에 대한 극좌표를 사용하여 색상을 지정합니다.

rgb()
RGB 색상 모델은 빨강, 녹색 및 파랑 구성 요소에 따라 주어진 색상을 정의합니다. 선택적 알파 구성 요소는 색상의 투명도를 나타냅니다.

rgba()
RGB 색상 모델은 빨강, 녹색 및 파랑 구성 요소에 따라 주어진 색상을 정의합니다. 알파 구성 요소는 색상의 투명도를 나타냅니다.

이미지 기능
이러한 함수는 <image>속성 값으로 유효한 모든 곳에서 사용할 수 있습니다.

conic-gradient()
원뿔형 그라디언트는 원을 중심으로 점진적으로 색상을 전환합니다.

image() 
<image>함수 와 유사한 방식으로 정의 url()하지만 이미지의 방향 지정, 기본 이미지가 지원되지 않는 경우에 대한 대체 이미지 지정 등의 기능이 추가되었습니다.

image-set() 
주로 높은 픽셀 밀도 화면에 대해 브라우저가 주어진 세트에서 가장 적절한 CSS 이미지를 선택하도록 하는 방법입니다.

linear-gradient()
선형 그라디언트는 가상의 선을 따라 점진적으로 색상을 전환합니다.

radial-gradient()
방사형 그라디언트는 중심점(원점)에서 점진적으로 색상을 전환합니다.

repeating-linear-gradiant()
와 유사 linear-gradient()하고 동일한 인수를 취하지만 전체 컨테이너를 덮기 위해 모든 방향에서 색상 정지를 무한히 반복합니다.

repeating-radial-gradient()
와 유사 radial-gradient()하고 동일한 인수를 취하지만 전체 컨테이너를 덮기 위해 모든 방향에서 색상 정지를 무한히 반복합니다.

repeat-conic-gradiant()
와 유사 conic-gradient()하고 동일한 인수를 취하지만 전체 컨테이너를 덮기 위해 모든 방향에서 색상 정지를 무한히 반복합니다.

cross-fade()

element()

글꼴 기능
속성 은 font-variant-alternates대체 글리프의 사용을 제어하며 다음 기능은 이 속성에 대한 값입니다.

stylistic()
이 기능은 개별 문자에 대한 문체 대체를 활성화합니다. 매개변수는 숫자에 매핑된 글꼴별 이름입니다. salt와 같은 OpenType 값에 해당합니다 salt 2.

styleset()
이 기능은 문자 집합에 대한 스타일 대안을 활성화합니다. 매개변수는 숫자에 매핑된 글꼴별 이름입니다. ssXY와 같은 OpenType 값에 해당합니다 ss02.

character-variant()
이 기능은 캐릭터에 대한 특정 스타일 대안을 활성화합니다. 와 비슷 styleset()하지만 문자 집합에 대해 일관된 글리프를 생성하지 않습니다. 개별 캐릭터는 독립적이고 반드시 일관된 스타일을 가질 필요는 없습니다. 매개변수는 숫자에 매핑된 글꼴별 이름입니다. cvXY와 같은 OpenType 값에 해당합니다 cv02.

swash()
이 기능은 튀기다글리프. 매개변수는 숫자에 매핑된 글꼴별 이름입니다. OpenType 값 swsh및 cswh, like swsh 2및 에 해당 cswh 2합니다.

ornaments()
이 기능은 다음과 같은 장식품을 활성화합니다. 플뢰론및 기타 dingbat 글리프. 매개변수는 숫자에 매핑된 글꼴별 이름입니다. ornm와 같은 OpenType 값에 해당합니다 ornm 2.

annotation()
이 기능은 원 숫자 또는 반전된 문자와 같은 주석을 활성화합니다. 매개변수는 숫자에 매핑된 글꼴별 이름입니다. nalt와 같은 OpenType 값에 해당합니다 nalt 2.
